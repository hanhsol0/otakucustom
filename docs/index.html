Otaku Custom Repository

Custom Otaku repository with Real-Debrid cache checking and UHD support.

Installation

  1. In Kodi, go to Settings → File Manager → Add Source
  2. Enter: https://hanhsol0.github.io/OtakuCustom/
  3. Name it "Otaku Custom" and click OK
  4. Go to Add-ons → Install from zip file → Otaku Custom
  5. Select repository.otakucustom-1.0.zip
  6. Then install addons from the repository
